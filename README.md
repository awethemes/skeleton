# Skeleton

A complete framework for WordPress, base on top of [CMB2](https://github.com/CMB2/CMB2).

## Features

- Metabox any where.
- Custom post type.
- Custom taxonomy.
- Page settings.

## Installation

Install via [Composer](https://getcomposer.org)

```
composer require awethemes/skeleton
```
